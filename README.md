## Perguntas Frequentes

**Como adiciono um disco à minha coleção?**
Você pode adicionar discos de três formas: digitando o nome do álbum ou artista, fotografando a capa do disco (a IA identifica automaticamente) ou escaneando o código de barras.

**Os preços de mercado são atualizados automaticamente?**
Sim. Os preços são obtidos via API do Discogs e podem ser atualizados manualmente a qualquer momento na tela de detalhes do disco.

**Minha coleção fica salva se eu trocar de iPhone?**
Sim. A coleção é sincronizada em nuvem e fica disponível em qualquer dispositivo onde você fizer login com a mesma conta.

**O app funciona sem internet?**
Sim. O app possui modo offline e exibe sua coleção a partir do cache local. As informações são sincronizadas automaticamente quando a conexão for restabelecida.

**Como faço backup da minha coleção?**
Na aba Perfil, na seção Backup, você pode exportar sua coleção para o iCloud Drive, Google Drive ou qualquer outro destino via compartilhamento nativo do iOS.

**Como cancelo minha conta?**
Para solicitar a exclusão da sua conta e de todos os dados associados, envie um e-mail para **filipe@seudominio.com** com o assunto "Excluir conta". Os dados serão removidos em até 30 dias.

## Política de Privacidade

O Vinyl Collector coleta apenas os dados necessários para o funcionamento do app:

- **Dados de conta:** e-mail e nome fornecidos no login (via OAuth)
- **Dados da coleção:** informações dos discos adicionados pelo usuário
- **Imagens:** fotos de capas enviadas pelo usuário para identificação via IA

Nenhum dado é vendido ou compartilhado com terceiros. As imagens enviadas para identificação são processadas e descartadas após o reconhecimento. Os dados da coleção são armazenados de forma segura em servidores protegidos.

As informações de álbuns são obtidas da API pública do Discogs. As imagens de capa são propriedade de seus respectivos detentores.

Para exercer seus direitos de acesso, correção ou exclusão de dados, entre em contato pelo e-mail acima.
